# Installation

- Python 3.13 or newer is required.

## From Source (development)

```
git clone https://github.com/mhbxyz/dictdb.git
cd dictdb
uv sync --group dev
```

Alternatively with pip:

```
pip install -e .
pip install pytest ruff mypy coverage
```

## From PyPI

Once published:

```
pip install dictdb
```
